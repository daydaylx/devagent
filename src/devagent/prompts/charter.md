# DevAgent CLI Charter

DevAgent CLI is an AI-powered development tool that helps developers make precise code changes through intelligent analysis and automated patch generation.

## Core Mission
- Scan and understand codebases comprehensively
- Provide AI-driven code change proposals
- Apply changes safely with validation and rollback capabilities
- Integrate seamlessly with existing development workflows

## Principles
- **Safety First:** All changes go through validation and can be rolled back
- **Precision:** Focus on minimal, targeted changes that solve specific problems
- **Context-Aware:** Leverage full project understanding for better decisions
- **Developer-Friendly:** Clear feedback, helpful errors, and intuitive workflows

## Workflow
1.  **Scan:** Build comprehensive project index and summaries
2.  **Check:** Validate project health (TypeScript, Python, tests, linting)
3.  **Propose:** Generate AI-driven change proposals with unified diffs
4.  **Apply:** Safely apply changes with validation and optional auto-commit
